# 🚀 AI 配置快速指南

## 三步快速配置

### 第一步：获取 AI API Key（5 分钟）

**推荐使用百度飞桨 AI Studio（完全免费）**

1. 访问 [百度 AI Studio](https://aistudio.baidu.com)
2. 点击右上角"登录"，用邮箱或手机号注册
3. 登录后点击"个人中心" → "Access Token"
4. 点击"获取 Access Token"按钮，复制生成的 Token

✅ **完成！** 你现在有了免费的 AI API Key

---

### 第二步：启动应用

启动应用后，主页会自动检测你的 AI 配置状态。

**如果看到弹窗提示 "⚠️ AI 服务未配置"**：

- 点击"去配置"按钮进入配置页面
- 或者手动点击主页的"AI 配置"按钮

---

### 第三步：填写 AI 配置

在 **AI 配置页面** 的 **"自定义服务"** 标签页中：

#### 视觉模型配置（"你画 AI 猜"需要）

| 项目 | 值 |
|------|-----|
| **API URL** | `https://aistudio.baidu.com/llm/lmapi/v3` |
| **API Key** | 粘贴第一步复制的 Token |
| **模型名称** | `ernie-4.5-vl-28b-a3b` |

然后点击"测试视觉模型"验证配置

#### 文生图模型配置（"AI 画你猜"需要）

| 项目 | 值 |
|------|-----|
| **API URL** | `https://aistudio.baidu.com/llm/lmapi/v3` |
| **API Key** | 粘贴第一步复制的 Token |
| **模型名称** | `Stable-Diffusion-XL` |

然后点击"测试文生图模型"验证配置

---

## 💡 常见问题

### Q: 为什么需要配置 AI 服务？

**A:** 本项目不包含本地大模型推理，所有 AI 功能都通过云 API 提供。这样可以保证：
- ⚡ 快速响应（毫秒级推理）
- 🎯 高准确度（使用业界最强模型）
- 💰 成本低廉（百度提供免费额度）

### Q: 百度 AI Studio 真的免费吗？

**A:** 完全免费！
- 注册后立即赠送 **100 万 tokens/月** 免费额度
- 足够普通用户使用
- 可用额度查询：登录后在"个人中心"查看

### Q: 还有其他 AI 服务选项吗？

**A:** 可以！本应用支持任何 **OpenAI 兼容格式的 API**，例如：
- OpenAI GPT-4V（需付费）
- Claude Vision（需付费）
- 其他兼容服务

配置方式相同，只需填入对应的 URL 和 Key 即可。

### Q: 测试连接失败怎么办？

**A:** 检查以下几点：
1. ✓ API Key 是否正确复制（没有多余空格）
2. ✓ 网络连接是否正常
3. ✓ 模型名称是否正确（区分大小写）
4. ✓ 百度后台是否有扣费/额度用完

### Q: 我用了自己的 AI API 会怎样？

**A:** 
- ✅ 完全支持
- ✅ API 调用费用由你的账户承担
- ✅ 配置私密保存在本地浏览器
